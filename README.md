# Interactive Math Counter (React 19 + Vite 8)

Современное интерактивное приложение-счетчик.

## ⚡️ Функционал
* **Арифметические операции:** Сложение, вычитание, умножение и деление на фиксированный шаг (5).
* **Динамический дизайн:** Яркий градиентный фон и прозрачные стеклянные (glassmorphism) элементы.
* **Умная индикация:** Число автоматически меняет цвет на красный при уходе в отрицательные значения.
* **Точность:** Все расчеты отображаются с точностью до двух знаков после запятой.

## 🛠 Стек технологий
* **React 19** — новейшая версия с поддержкой **React Compiler**.
* **Vite 8** — максимально быстрая сборка.
* **Tailwind CSS v4** — современная стилизация через новые движки.
* **TypeScript** — полная типизация компонентов.
